Recent Advances in Few-Shot Learning

Posted on Thu 02 April 2020 in posts

Multiple recent papers suggest a simpler baseline for few-shot classification that perform better than sophisticated meta-learning algorithms.

  1. Train a network from scratch by merging all the meta-training tasks into one dataset.
  2. Use this network as a fixed feature extractor for tasks in the meta-test set.

Papers:

  • A Baseline for Few-Shot Image Classification (ICLR 2020) [Paper]
  • Re-thinking Few-Shot Image Classification: a Good Embedding Is All You Need? (Josh Tenenbaum, Philip Isola) [Paper] [Code]
  • A New Meta-Baseline for Few-Shot Learning (Trevor Darrell) [Paper] [Code]

Another common theme of these papers is the importance of the feature embedding/network weights obtained during the meta-training process. (Related paper)

Datasets:

  1. CIFAR-FS
  2. FC100
  3. miniImageNet
  4. tieredImageNet
  5. Omniglot
  6. Meta-Dataset
