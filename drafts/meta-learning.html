Metric Based

These approaches perform few-shot classification by comparing how similar the query image is to all the images in the support set. The approach is similar to KNN classification.

One Shot Recognition

A siamese network is trained to output the probability of two images being in the same class. This is called verification. At test time, the query image is compared to one image from each class and the class of the image with the highest similarity is given as the output.
